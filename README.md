# Random Alphanumeric Generator

A simple Node.js utility function to generate random alphanumeric strings of custom length.

---

## Features

* Generate random strings
* Custom length support
* Alphanumeric characters (`A-Z`, `a-z`, `0-9`)
* Lightweight and easy to use
* Secure version using Node.js `crypto`

---

## Installation

Clone the repository:

```bash
git clone https://github.com/avibhardwaj243/randomNumberGeneratorNpmPackage
```

Go to project folder:

```bash
cd randomNumberGeneratorNpmPackage
```
---

## Parameters

| Parameter | Type   | Description             |
| --------- | ------ | ----------------------- |
| length    | Number | Length of random string |
